# Template Profiles

Use one of these template profiles for a cleaner academic poster result.

## `conference-classic`

- standard academic conference poster
- top title banner
- left column: background and method
- center column: results and main figures
- right column: conclusion, highlights, references
- best default choice for teacher demo

## `data-focus`

- for machine learning, data science, engineering experiments
- large central results zone
- fewer text blocks
- figure-first composition

## `compact-journal`

- more text-dense but still controlled
- suitable for literature review or social science summary
- smaller figures, stronger section hierarchy

## `visual-showcase`

- for papers with strong charts, microscopy, diagrams, interfaces, or workflows
- fewer words
- more emphasis on figure panels

## Recommendation

If the user does not specify a template, default to `conference-classic`.

## Quality Rule

For poster figures, prefer:

- at least 1400 px on the long edge for a major figure
- at least 1000 px on the long edge for a secondary figure
- no visibly blurred screenshots unless the user explicitly accepts them
